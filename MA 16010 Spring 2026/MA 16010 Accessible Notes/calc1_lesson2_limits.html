Applied Calculus I - Lesson 2: Finding Limits Numerically; One-sided Limits; Finding Limits Graphically

Limits

Definition: The limit of a function \(f(x)\) is the value that \(f(x)\) approaches as \(x\) approaches a particular value. If \(f(x)\) approaches \(L\) as \(x\) approaches \(c\), we say that the limit of \(f(x)\) as \(x\) approaches \(c\) is \(L\), denoted by

\[\lim_{x \to c} f(x) = L.\]

Important: The function \(f(x)\) does not need to be defined at \(x = c\) for the limit to exist.

Definition: If \(f(x)\) increases or decreases without bound as \(x\) approaches \(c\), then \(\lim_{x \to c} f(x)\) is an infinite limit. If \(f(x)\) increases without bound, \(\lim_{x \to c} f(x) = \infty\). If \(f(x)\) decreases without bound, \(\lim_{x \to c} f(x) = -\infty\).

Important: A limit does not need to exist, that is, a limit can evaluate to DNE (does not exist). This happens when \(f(x)\) does not approach a specific value as \(x\) approaches \(c\).

Finding Limits Numerically

To estimate \(\lim_{x \to c} f(x)\), we can evaluate \(f(x)\) at values of \(x\) which get closer and closer to \(c\). This approach involves creating a table of values that approach \(c\) from both the left (values less than \(c\)) and the right (values greater than \(c\)), and observing what value the function outputs are approaching.

Example 1: Linear Function Limit

Evaluate the following limit numerically:

\[\lim_{x \to 1} (x + 3) = ?\]
\(x\) 0.9 0.99 0.999 0.9999 1 1.0001 1.001 1.01 1.1
\(f(x)\) 3.9 3.99 3.999 3.9999 — 4.0001 4.001 4.01 4.1

Solution:

As \(x\) approaches 1 from both the left and right, \(f(x) = x + 3\) approaches 4. So,

\[\lim_{x \to 1} (x + 3) = 4.\]

Notice that \(1 + 3 = 4\), so the function is defined at \(x = 1\) and equals the limit value.

Example 2: Rational Function Limit

Evaluate the following limit numerically:

\[\lim_{x \to 2} \frac{x^2 - 2x}{x - 2} = ?\]
\(x\) 1.9 1.99 1.999 1.9999 2 2.0001 2.001 2.01 2.1
\(f(x)\) 1.9 1.99 1.999 1.9999 — 2.0001 2.001 2.01 2.1

Solution:

As \(x\) approaches 2 from both sides, the function values approach 2. Therefore,

\[\lim_{x \to 2} \frac{x^2 - 2x}{x - 2} = 2.\]

Notice that \(\frac{x^2 - 2x}{x - 2}\) is not defined at \(x = 2\) because the denominator becomes zero. However, the limit still exists. This example illustrates that a function does not need to be defined at a point for the limit to exist at that point.

Example 3: Infinite Limit

Evaluate the following limit numerically:

\[\lim_{x \to -4} \frac{8}{(x + 4)^2} = ?\]
\(x\) -4.1 -4.01 -4.001 -4.0001 -4 -3.9999 -3.999 -3.99 -3.9
\(f(x)\) 800 80000 8000000 800000000 — 800000000 8000000 80000 800

Solution:

As \(x\) approaches \(-4\) from both sides, the function values increase without bound. The function values grow larger and larger as \(x\) gets closer to \(-4\), indicating an infinite limit. Therefore,

\[\lim_{x \to -4} \frac{8}{(x + 4)^2} = \infty.\]

Notice that \(\frac{8}{(x + 4)^2}\) is not defined at \(x = -4\).

One-sided Limits

Definition: A one-sided limit is the value a function \(f(x)\) approaches as \(x\) approaches a particular value from the left or right.

Definition (left-sided limit): If \(f(x)\) approaches \(L\) as \(x\) approaches \(c\) from the left, we write

\[\lim_{x \to c^-} f(x) = L.\]

Definition (right-sided limit): If \(f(x)\) approaches \(L\) as \(x\) approaches \(c\) from the right, we write

\[\lim_{x \to c^+} f(x) = L.\]
Example 4: Piecewise Function with One-sided Limits

Suppose

\[f(x) = \begin{cases} 5 & \text{if } x > 1, \\ x & \text{if } x \leq 1. \end{cases}\]

Evaluate the following limits numerically:

\[\lim_{x \to 1^-} f(x) = ? \qquad\qquad \lim_{x \to 1^+} f(x) = ? \qquad\qquad \lim_{x \to 1} f(x) = ?\]
\(x\) 0.9 0.99 0.999 0.9999 1 1.0001 1.001 1.01 1.1
\(f(x)\) 0.9 0.99 0.999 0.9999 1 5 5 5 5

Solution:

From the table, we can observe the behavior of \(f(x)\) as \(x\) approaches 1 from both directions.

As \(x\) approaches 1 from the left (values less than 1), \(f(x) = x\) approaches 1. Therefore,

\[\lim_{x \to 1^-} f(x) = 1.\]

As \(x\) approaches 1 from the right (values greater than 1), \(f(x) = 5\) for all such values. Therefore,

\[\lim_{x \to 1^+} f(x) = 5.\]

Since the left-sided limit (1) and the right-sided limit (5) are different, the two-sided limit does not exist. Therefore,

\[\lim_{x \to 1} f(x) = \text{DNE}.\]

Theorem: The two-sided limit exists if and only if both one-sided limits exist and are equal. Specifically,

\[\lim_{x \to c} f(x) = L \text{ if and only if } \lim_{x \to c^-} f(x) = L = \lim_{x \to c^+} f(x).\]

Note that we allow \(L = \pm\infty\).

Finding Limits Graphically

When evaluating limits graphically, we look at the portion of the curve of \(f(x)\) near \(x = c\) and observe what the function value approaches as \(x\) gets closer to \(c\), from both the left and right. The key is to focus on the behavior of the function near the point, not necessarily the value of the function at the point itself.

Example 5: Multiple Limits from a Graph

Graph Description: The graph shows a piecewise function \(f(x)\) on a coordinate plane with the following features: At \(x = -4\), there is an open circle at \(y = 4\) and a closed circle at \(y = 3\). As \(x\) approaches -4 from the left, the function approaches \(y = 3\). As \(x\) approaches -4 from the right, the function approaches \(y = 4\). The function approaches \(y = 2\) from both sides as \(x\) approaches \(-1\), but the actual function value at \(x = -1\) is -1. At \(x = 3\), there is an open circle at \(y = -3\) and a closed circle at \(y = -2\). As \(x\) approaches 3 from the left, the function approaches \(y = -3\). As \(x\) approaches 3 from the right, the function approaches \(y = -2\).

Fill in the following table:

Limit Expression At \(x = -4\) At \(x = -1\) At \(x = 3\)
\(\lim_{x \to c^-} f(x)\)
\(\lim_{x \to c^+} f(x)\)
\(\lim_{x \to c} f(x)\)
\(f(c)\)

Solution:

For \(x = -4\): The left-sided limit is 3, but the right-sided limit is 4. Therefore, the two-sided limit does not exist. Note that \(f(-4)=3\).

For \(x = -1\): Both one-sided limits equal 2, implying that the two-sided limit is 2. Note that \(f(-1)=-1\).

For \(x = 3\): The left-sided limit is -3, but the right-sided limit is -2. Therefore, the two-sided limit does not exist. Note that \(f(3)=-2\).

Example 6: Infinite Limits

Graph Description: As \(x\) approaches 0 from the left, the function decreases without bound. As \(x\) approaches 0 from the right, the function increases without bound. The function is not defined at \(x = 0\).

Find the following using the graph of \(f(x)\):

\[\lim_{x \to 0^-} f(x) = ? \qquad \lim_{x \to 0^+} f(x) = ? \qquad \lim_{x \to 0} f(x) = ? \qquad f(0) = ?\]

Solution:

From the graph, we observe:

\[\lim_{x \to 0^-} f(x) = -\infty\] \[\lim_{x \to 0^+} f(x) = \infty\]

Since the one-sided limits are not equal:

\[\lim_{x \to 0} f(x) = \text{DNE}\]

The function is undefined at \(x=0\).

Important: When finding limits graphically, remember to distinguish between the limit of a function at a point and the value of the function at that point. The limit depends on the behavior of the function near the point, while the function value depends on whether the point is included in the graph (closed circle) or not (open circle).
