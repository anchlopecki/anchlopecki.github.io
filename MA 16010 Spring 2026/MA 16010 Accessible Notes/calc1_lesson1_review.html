Applied Calculus I - Lesson 1: Review

This lesson reviews fundamental mathematical concepts that form the foundation for calculus, including exponential functions, logarithms, and trigonometry.

Exponentials

Let \(a\), \(m\), and \(n\) be real numbers. The properties of exponential functions govern how we manipulate expressions involving powers and exponents.

Properties of Exponentials:

  • \(a^m \cdot a^n = a^{m+n}\)
  • \(\frac{a^m}{a^n} = a^{m-n}\)
  • \((a^m)^n = a^{mn}\)
  • \(a^{-m} = \frac{1}{a^m}\)
  • \(a^0 = 1\), if \(a \neq 0\)

Logarithms

Let \(a\) and \(b\) be real numbers, and let \(c > 0\). The logarithm function with base \(c\), denoted \(\log_c(x)\), is the inverse of the exponential function with base \(c\).

Properties of Logarithms:

  • \(\log_c(ab) = \log_c(a) + \log_c(b)\)
  • \(\log_c\left(\frac{a}{b}\right) = \log_c(a) - \log_c(b)\)
  • \(\log_c(a^b) = b \log_c(a)\)
  • \(\log_c(c) = 1\)
  • \(\log_c(1) = 0\)

Natural Logarithm

Definition: The natural logarithm function is defined as \(\log_e(x) = \ln(x)\), where \(e\) is Euler's number (approximately 2.71828).

Applying the general logarithm properties to the natural logarithm yields:

  • \(\ln(ab) = \ln(a) + \ln(b)\)
  • \(\ln\left(\frac{a}{b}\right) = \ln(a) - \ln(b)\)
  • \(\ln(a^b) = b \ln(a)\)
  • \(\ln(e) = 1\)
  • \(\ln(1) = 0\)

Important: In this course, we will primarily be using the natural logarithm function. Note that \(e^{\ln(x)} = x\) and \(\ln(e^x)=x\).

Useful Identities

The following identities relate the six fundamental trigonometric functions:

Trigonometric Identities:

  • \(\tan(x) = \frac{\sin(x)}{\cos(x)}\)
  • \(\cot(x) = \frac{\cos(x)}{\sin(x)}\)
  • \(\sec(x) = \frac{1}{\cos(x)}\)
  • \(\csc(x) = \frac{1}{\sin(x)}\)
  • \(\sin^2(x) + \cos^2(x) = 1\)

Practice Problems

Problem 1: Simplify the following expressions.

(a) \(e^2 \cdot e^3\)

Solution: Using the property \(a^m \cdot a^n = a^{m+n}\), we have:

\[e^2 \cdot e^3 = e^{2+3} = e^5\]

(b) \((e^2)^3\)

Solution: Using the property \((a^m)^n = a^{mn}\), we have:

\[(e^2)^3 = e^{2 \cdot 3} = e^6\]

(c) \(e^{-8} \cdot e^m\)

Solution: Using the property \(a^m \cdot a^n = a^{m+n}\), we have:

\[e^{-8} \cdot e^m = e^{-8+m} = e^{m-8}\]

(d) \(\frac{e^{2x} \cdot e^y}{e^z}\)

Solution: First, combine the numerator using \(a^m \cdot a^n = a^{m+n}\), then apply \(\frac{a^m}{a^n} = a^{m-n}\):

\[\frac{e^{2x} \cdot e^y}{e^z} = \frac{e^{2x+y}}{e^z} = e^{2x+y-z}\]

(e) \(e^{\ln(5x)}\)

Solution: Since \(e^{\ln(x)} = x\), we have:

\[e^{\ln(5x)} = 5x\]

(f) \(e^{10-2\ln(5)}\)

Solution: We can rewrite this using properties of exponents and logarithms:

\[e^{10-2\ln(5)} = e^{10} \cdot e^{-2\ln(5)} = e^{10} \cdot e^{\ln(5^{-2})} = e^{10} \cdot 5^{-2} = \frac{e^{10}}{25}\]
Problem 2: Solve for x.

Solve the following equation for \(x\):

\[\ln(x^2) = 20\]

Solution: We note the following trick:

\[e^{\ln(x^2)}=e^{20}\]

Using the fact that \(e^{\ln(x)}=x\):

\[x^2=e^{20}\]

Taking the square root of both sides:

\[x=\pm \sqrt{e^{20}}=\pm (e^{20})^{1/2}=\pm e^{10}\]
Problem 3: Express as sums, differences, and multiples of basic logarithms.

(a) \(\ln\left(\frac{xy}{z}\right)\)

Solution: Using logarithm properties:

\[\ln\left(\frac{xy}{z}\right) = \ln(xy) - \ln(z) = \ln(x) + \ln(y) - \ln(z)\]

(b) \(\ln\left(\frac{x^2}{y\sqrt{z}}\right)\)

Solution: First, rewrite \(\sqrt{z} = z^{1/2}\), then apply logarithm properties:

\[\ln\left(\frac{x^2}{y\sqrt{z}}\right) = \ln(x^2) - \ln(y\sqrt{z}) = \ln(x^2) - \ln(y) - \ln(z^{1/2}) = 2\ln(x) - \ln(y) - \frac{1}{2}\ln(z)\]
Problem 4: Find the trigonometric values from the given information.

Suppose \(\cos(\theta) = \frac{2}{3}\), and \(\theta\) is in the fourth quadrant. Compute \(\sin(\theta)\), \(\cot(\theta)\), and \(\sec(\theta)\).

Solution: We use the Pythagorean identity \(\sin^2(\theta) + \cos^2(\theta) = 1\) to find \(\sin(\theta)\):

\[\sin^2(\theta) + \left(\frac{2}{3}\right)^2 = 1\] \[\sin^2(\theta) + \frac{4}{9} = 1\] \[\sin^2(\theta) = 1 - \frac{4}{9} = \frac{5}{9}\] \[\sin(\theta) = \pm\frac{\sqrt{5}}{3}\]

Since \(\theta\) is in the fourth quadrant where sine is negative:

\[\sin(\theta) = -\frac{\sqrt{5}}{3}\]

Now we can find \(\cot(\theta)\) using \(\cot(\theta) = \frac{\cos(\theta)}{\sin(\theta)}\):

\[\cot(\theta) = \frac{\frac{2}{3}}{-\frac{\sqrt{5}}{3}} = -\frac{2}{\sqrt{5}}\]

Finally, we find \(\sec(\theta)\) using \(\sec(\theta) = \frac{1}{\cos(\theta)}\):

\[\sec(\theta) = \frac{1}{\frac{2}{3}} = \frac{3}{2}\]
